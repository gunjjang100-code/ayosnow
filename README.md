# 웹작업 폴더 안내

이 폴더는 두 작업을 분리해서 보관합니다.

## AyosNow

기존 홈서비스/마켓플레이스 프로젝트입니다.

작업할 때는 이 폴더로 들어갑니다.

```bash
cd AyosNow
npm run dev
```

## HeartTalk-AI

현재 새로 만드는 AI 연애 코치 + 타로 + 채팅 MVP입니다.

작업할 때는 이 폴더로 들어갑니다.

```bash
cd HeartTalk-AI
npm run dev
```

## 루트에 남은 폴더

- `.git`: Git 기록 보관 폴더입니다.
- `.gitignore`: Git에 올리지 않을 파일 규칙입니다.

쉽게 말하면, `AyosNow`와 `HeartTalk-AI`는 각각 따로 놓인 작업 책상입니다.
